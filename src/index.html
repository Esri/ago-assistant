ArcGIS Online Assistant

A swiss army knife for your ArcGIS Online and ArcGIS Enterprise accounts.

Inspect

View the underlying JSON for any item in ArcGIS Online or ArcGIS Enterprise.

Update

Modify the URLs for services in web maps and registered applications.

Copy

Search for and copy content to your account and other accounts.

About this application

v changelog

This app uses the ArcGIS REST API to help you work with your ArcGIS Online and ArcGIS Enterprise accounts and content.

The code is available on GitHub. Please contribute to this project by submitting ideas, feedback, issues, or code.

Sign in to ArcGIS Enterprise

Enter the URL to your ArcGIS Enterprise portal


  • Direct Login
  • OAuth Login
  • PKI or IWA Login

More info
This login method makes use of a direct call to the GenerateToken operation of the ArcGIS REST API in your ArcGIS Enterprise portal. Username and Password are case-sensitive.

More info
In order to sign in to ArcGIS Enterprise using a SAML-based Identity Provider, you will need to Register AGO-Assistant as an application in your Enterprise portal, to generate an AppID that can identify this app as an allowed client of that Enterprise portal. To do so, follow the instructions here, using Application as the Type of App and as the Redirect URI.

More info
This login method is used for ArcGIS Enterprise portals with web-tier authentication, using username and password or client certificate (PKI) authentication methods, often referred to as “SSO” or Single Sign On). This authentication method is most often used for internal-facing ArcGIS Enterprise setups inside of an intranet. More info

Copy information

Simple copy creates a reference to the original service in the destination account.

Full copy (EXPERIMENTAL) replicates the original service in the destination account and harvests all of its associated data.
